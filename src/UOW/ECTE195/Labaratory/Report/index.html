Discussion on this web page

  • This webpage was done in VI since Composer in Netscape wouldn't know it's <head> from it's </head>.
  • SGML like tags were used for increased portability between browsers.
  • A monochrome look was adopted for those familiar with CRO's.

A table of the HTML Tags used
